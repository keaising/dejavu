
订单/Order
=====================

创建订单
*********************

接口
+++++++++++++++++++++

/order/create

请求类型
+++++++++++++++++++++

POST

传入参数
+++++++++++++++++++++

.. table::

    +-------------+------+--------------------------------+------------+
    |    name     | type |              e.g.              |is_necessary|
    +=============+======+================================+============+
    |address_id   |string|                     15522229999|True        |
    +-------------+------+--------------------------------+------------+
    |fee          |string|password123*                    |True        |
    +-------------+------+--------------------------------+------------+
    |book_id_price|string|id1:price1,id2:price2,id3:price3|True        |
    +-------------+------+--------------------------------+------------+



返回值
+++++++++++++++++++++

.. code:: json

    {
        "code": 200,
        "msg": "success",
        "data": null
    }

失败：

.. code:: json

    {
        "code": 400,
        "msg": "这里会有具体的失败原因",
        "data": null
    }

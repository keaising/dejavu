
账户/Account
=====================

查询用户账户余额
*********************

接口
+++++++++++++++++++++

/account/surplus

请求类型
+++++++++++++++++++++

GET

传入参数
+++++++++++++++++++++

无（从cookie中获取用户信息）


返回值
+++++++++++++++++++++

.. code:: json

    {
        "code": 200,
        "msg": "success",
        "data": {
            "mobile":"155111116666",
            "surplus":233.33,
        }
    }

失败：

.. code:: json

    {
        "code": 400,
        "msg": "这里会有具体的失败原因",
        "data": null
    }

充值
*********************

接口
+++++++++++++++++++++

/account/recharge

请求类型
+++++++++++++++++++++

POST

传入参数
+++++++++++++++++++++

.. table::

    +----------+-------+------------+------------+
    |   name   | type  |    e.g.    |is_necessary|
    +==========+=======+============+============+
    |mobile    |string | 15522229999|True        |
    +----------+-------+------------+------------+
    |fee       |decimal|2333        |True        |
    +----------+-------+------------+------------+


返回值
+++++++++++++++++++++

.. code:: json

    {
        "code": 200,
        "msg": "success",
        "data": null
    }

失败：

.. code:: json

    {
        "code": 400,
        "msg": "这里会有具体的失败原因",
        "data": null
    }

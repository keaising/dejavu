购物车/Cart 接口¶

查询特定用户购物车¶

接口¶

/cart/query

请求类型¶

POST

传入参数¶

无（从cookie中获取用户信息）

返回值¶

{
    "code": 200,
    "msg": "success",
    "data": [
        {
            "id": 1,
            "mobile": "15511118888",
            "book_id": "233",
            "count": 2,
            "add_date": "2019-10-06T17:12:40",
        }
    ]
}

失败：

{
    "code": 400,
    "msg": "这里会有具体的失败原因",
    "data": null
}
